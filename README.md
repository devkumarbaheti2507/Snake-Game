# Snake Game in C++

A simple Snake game built using C++ for Windows console. Features:
- Dynamic snake movement
- Collision detection
- Score tracking
- Fruit generation

## Controls

- `W` = Up
- `S` = Down
- `A` = Left
- `D` = Right
- `X` = Exit Game

## How to Run

1. Compile the code:
   ```bash
   g++ snake-game.cpp -o snake-game
